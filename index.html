Web Apps

Anti-Piracy App

The Anti-Piracy App serves the purpose of "authorizing" another App to run on a specific device running a specific browser.

The App is able to "stamps" an alphameric string into the localStorage of the browser that runs it. The string is defined by the user.

Once launched, the Anti-Piracy App fetches and displays the existing stamp from local storage, if any. It can be used to create stamp, read stamp, erase stamp, or modify stamp.

Once local storage is "stamped", any other App can be designed to fetch the stamp, and compare against its internal hard-coded alphameric string. A match would mean App is authorized to run on the device. A no match prevents that App from running.

Rolling Door Spring Selector

Before running this App, you must run the Anti-Piracy App above to authorize the device (PC/cellphone) that will run the App. Use any desired alphameric string on the Anti-Piracy App.

The Rolling Door Spring Selector is a real world App I sold to a customer. It is useful to repair or to design Rolling Doors. It allows the user to specify all rolling door relevant components in order to determine the spring to counterbalance the resulting curtain weight.

Application note:
This is two Apps in one. It can determine optimal helicoidal torsion spring, or optimal set of band springs (metal ribbon style springs).
That can be achieved by selecting either "Barrel Style" 4 or 5 on the "Barrel" page; which is the 3rd page of the App.
On both cases the App uses a pre-defined set of springs (the inventory).

In the case of torsion springs, the App allows selection of the pre-defined inventory, or the whole available spring wire sizes that the spring manufacturer has available.
